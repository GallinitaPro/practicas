Atributos y su función

Atributo

Función
Align Se utilizaba para controlar la alineación horizontal de elementos, como texto y imágenes. Aunque ya no es compatible con HTML5, en el pasado permitía alinear elementos a la izquierda, derecha o centro. En HTML5, se recomienda utilizar propiedades CSS para controlar la alineación horizontal del texto y elementos.
Valign Se utiliza para definir la alineación vertical del contenido dentro de una celda de tabla. Controla la posición vertical del texto, imágenes u otros elementos dentro de la celda, permitiendo alinear el contenido en la parte superior, centro o inferior de la celda, o incluso basándose en la línea base del texto.
Background Controla el fondo de un elemento. Puede definir un color de fondo o una imagen para el cuerpo de la página web o para otros elementos como tablas.
Bgcolor Solía ser usado para definir el color de fondo de un elemento. Sin embargo, actualmente está obsoleto y se recomienda usar la propiedad CSS en su lugar.
Bodercolor Se utiliza para definir el color de los bordes de una tabla. Este atributo ha sido deprecado en HTML5, pero aún se puede encontrar en versiones más antiguas de HTML y en otros contextos donde se utiliza para configurar la apariencia visual de las tablas.
Cellpadding Controla el espacio entre el borde de una celda y su contenido.Especifica la cantidad de espacio, en píxeles, que habrá entre el borde de la celda y lo que se encuentra dentro de ella, como texto, imágenes u otros elementos.
Cellspacing Controla el espacio entre los bordes de celdas adyacentes en una tabla. Define la distancia entre los bordes de las celdas, permitiendo separarlas visualmente. Este atributo es una característica de las tablas HTML, aunque en HTML5 y CSS moderno, se prefiere el uso de la propiedad border-spacing para lograr el mismo efecto.
Border Se utiliza para definir el grosor del borde de un elemento, generalmente en tablas. En HTML, el atributo border se usa para especificar el ancho del borde alrededor de un elemento. También, en CSS, la propiedad border controla el grosor, estilo y color de los bordes de un elemento.
Height Define la altura de un elemento HTML en píxeles. Puede utilizarse en diferentes elementos HTML, como la etiqueta para especificar la altura de la imagen, o en otros elementos para establecer su altura vertical.
Width Se utiliza para definir el ancho de un elemento en HTML, especificando la cantidad de espacio horizontal que ocupa el elemento en píxeles. En el contexto de imágenes, especifica el ancho de visualización de la imagen.
